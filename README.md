# PokeFauna

Este repositorio contiene varios scripts y notebooks utilizados en la creación de una Pokédex con reconocimiento de imágenes. Para ejecutar los scripts es necesario instalar las dependencias de Python.

## Instalación

Primero, se recomienda crear un entorno virtual (opcional):

```bash
python -m venv venv
source venv/bin/activate
```

Instala las dependencias con:

```bash
pip install -r requirements.txt
```

Con esto podrás ejecutar los notebooks y las aplicaciones de ejemplo que están en las carpetas `POC` y `webapp`.
